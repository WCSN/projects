 Вероятности в 1000.
Начну с начала, т.е. с предельно общих моментов. В тысячу играют колодой из 24 карт, всего по 6 карт каждой масти (Т 10 К Д В 9). Далее речь идет об игре втроем или вчетвером (когда один из игроков сидит на прикупе). Троим игрокам сдается по 7 карт, оставшиеся 3 карты ложатся в прикуп. Партия играется или в темную (двойной счет, но при этом заказывающий игрок не видит своих карт до момента взятия прикупа), или в светлую (игрок смотрит свои 7 карт и решает, торговаться за прикуп или нет). 
Число сочетаний.

Сколькими способами можно выбрать 1 карту из 4? – Четырьмя различными способами.
Сколькими способами можно выбрать 2 карты из 4? Пронумеруем карты от 1 до 4. Возможные сочетания карт: 1 2, 1 3, 1 4, 2 3, 2 4, 3 4. Итого шестью различными способами. 
Число сочетаний из n по k (n>=k) равно количеству способов, которыми можно выбрать k элементов из n элементов без повторений, обозначается nCk.

nCk = n! / ( k! * (n-k)! ), (1.1)

где &#171;!&#187; - факториал числа, вычисляется как произведение всех чисел от данного числа до единицы (или наоборот, что не меняет сути дела). 5! = 5*4*3*2*1 = 120.
(*)Формула для нахождения числа сочетаний важна, т.к. все дальнейшие расчеты отталкиваются от нее.

Игра в темную.

Вероятность нахождения в прикупе определенных карт.

Для начала решим задачу о нахождении в прикупе некоторого количества нужных нам карт. Всего в колоде 24 карты, ни одна, ни хуже и не лучше любой другой, колода не заряжена, поэтому попадание любой карты в прикуп есть событие случайное и равновероятное для всех карт.

Вероятность попадания 1 любой карты в прикуп.
Число возможных прикупов определится 24C3 = 2024, т.е. сколькими способами можно выбрать 3 карты из 24. Далее, найдем число прикупов, не удовлетворяющих условию, т.е. нужная нам карта не вошла в прикуп. Число таких прикупов определится (24-1)С3 = 23С3 = 1771. Вычислим вероятность по следующей формуле:

p = (всего прикупов - число неудачных прикупов) / всего прикупов * 100&#37;. (1.2)
p(1,1) = (24C3-23C3)/24C3*100% = 12.50%. (1.3)

Вероятность, при игре в темную докупить 1 любую карту равна 12.50%.
Обобщая вышеприведенную формулу (1.3), получим формулу для расчета вероятности попадания в прикуп хотя бы одной из n нужных нам карт (n>=1):
p(1,n) = (24C3 - (24-n)C3) / 24C3 * 100%, где n=1..21. (1.4)
для n = 22, 23, 24 вероятность равна 100%, т.к. вычитаемое вырождается.

Пример. Вероятность докупить любого туза: p(1,4) = (24C3-20C3)/24C3*100% = 43.68%.

Вероятность докупить 2 карты из nнужных (n>=2).
Рассуждения совершенно аналогичны рассуждениям для одной карты, за тем исключением, что нам также не подходят и те прикупы, в которые вошла только одна из двух нужных нам карт. В числитель дроби добавится вычитаемое n*(24-n)C2. Формула для расчета вероятности:

p(2,n) = (24C3 - n*(24-n)C2 - (24-n)C3) / 24C3 * 100%, где n=2..22. (1.5)
для n = 23, 24 вероятность равна 100%, т.к. вычитаемые вырождаются.

Пример. Вероятность докупить любого туза с любой десяткой, или 2 любых туза, или 2 любые десятки. Речь идет о вероятности докупить 2 карты из 8 (4 Т + 4 10). p(2,8) = 24.90%. Удачным будет каждый четвертый прикуп.

Вероятность докупить 3 карты из nнужных (n>=3).
В данном случае проще посчитать число удачных прикупов и отнести его к общему числу прикупов (24С3).

p(3,n) = nC3 / 24C3 * 100%, где n=3..24. (1.6)

Задача 1.1. Вероятность докупить прикуп, сумма карт которого не ниже 20 очков.
В любом случае ни одна комбинация составленная из карт К Д В 9 не даст 20 очков. Минимальная комбинация: 10 10 9. Определили, что 2 любые карты из 3 должны быть или Т или 10, третья карта может быть любой. Т.е. первые 2 карты выбираются из 8 (4 Т + 4 10) и после того, как 2 карты заняли свое место в прикупе, оставшаяся карта выбирается из 22.
Искомая вероятность: p = 8C2*22/24С3*100% = 30.43%.

Задача 1.2. Вероятность докупить любую пару (хваль) (40, 60, 80, 100).
Всего есть 4 пары, каждая из которых &#171;займет&#187; 2 карты из 3. Оставшаяся карта может быть любой из 22
Искомая вероятность: p = 4*22/24С3*100% = 4.35%.

Задача 1.3. Вероятность докупить хотя бы 1 любую половину (К или Д любой масти).
Речь идет о вероятности докупить любую 1 карту из 8.
Искомая вероятность: p(1,8) = (24C3-16C3)/24C3*100% = 72.33%.

Данные вероятности скорее могут пригодиться игроку сидящему на прикупе. Для игрока заказывающего игру в темную ниже рассмотрим более общую задачу.

Игра в темную. Более общая задача.

После того как игрок заказал игру в темную, у него на руках оказываются 10 карт, 2 из которых он сносит своим оппонентам. Задача ставится следующим образом: рассчитать вероятность нахождения на руках m карт из n нужных (n>=m).
Число возможных раскладов - 24С10 (т.е. сколькими способами можно выбрать 10 карт из 24). Действуя по аналогии с задачей &#171;о нахождении в прикупе определенного числа карт&#187;, исключая неудачные для нас расклады из общего числа таковых, в конечном итоге придем к общей формуле (2.1). (Для того чтобы не перегружать вас лишними расчетами сразу приведу обобщенную формулу, без вывода).

p(m,n) = (24C10 - ∑[ nC(a-1) * (24-n)C(10-a+1) ] ) / 24C10 * 100%, (2.1)
где a=m..1. т.е. в каждом аргументе суммы значение переменной a последовательно уменьшается на единицу.

Задача 2.1. Вероятность получить на руки 3 любых туза при игре втемную.
Для решения данной задачи найдем частный случай формулы (2.1) для m=3.

p(3,n) = (24C10 - nC2*(24-n)C8 - n*(24-n)C9 - (24-n)C10) / 24C10 * 100%. (2.2)
(nC1 = n, nC0 = 1)
Всего в колоде 4 туза, поэтому искомая вероятность:
p(3,4) = 17.79%.

Задача 2.2. Вероятность наличия на руках хотя бы одной любой пары (хвали) (40, 60, 80, 100) при игре в темную.
Просто воспользоваться формулой (2.1) здесь не получится, т.к. при решении данной задачи имеет значение взаимное расположение карт. Интересующие нас половины (всего 8 карт) в расклад могут и войти, но пару могут не составлять.
По аналогии с предыдущими расчетами, найдем число раскладов, не удовлетворяющих условию задачи.

1) не вошло ни 1й половины: 16С10
2) вошла только 1 половина: 8*16С9
3) вошло 2 половины, но нет пары: 16С8*(8С4-4) = 24*16С8
4) вошло 3 половины, но нет пары: 16С7*(8С3-4*6) = 32*16С7
5) вошло 4 половины, но нет пары: 16С6*(8С4-4*(6С3-3)-4С2) = 16*16С6

При любой комбинации из 5, 6, 7, 8 любых половин, так или иначе, есть хотя бы 1 пара.
Искомая вероятность найдется: (общее число комбинаций - число неудачных комбинаций) / общее число комбинаций * 100%.

p = (24С10-16С10-8*16С9-24*16С8-32*16С7-16*16С6)/24C10*100% = 53.98%.

Задача 2.3. Вероятность наличия на руках хотя бы одной любой пары (40, 60, 80, 100) с хотя бы одним любым тузом при игре втемную.
Пойдем другой путёй. Посчитаем число комбинаций, удовлетворяющих условию задачи. Всего имеем дело с 12 интересующими нас картами.

1) вошло 3 карты, удачная комбинация половина + туз: 4*4*12С7 = 16*12С7
2) вошло 4 карты, число удачных комбинаций:
а) пара + туз + любая: 4*4*6 = 96
б) пара + 2 туза: 4*4С2 = 24
итого: (96+24)*12С6 = 120*12С6
3) вошло 5 карт, число удачных комбинаций:
а) пара + туз + 2 любых: 4*4*(6С2-3) = 192
б) пара + 2 туза + любая: 4*4С2*6 = 144
в) пара + 3 туза: 4*4С3 = 16
г) 2 пары + туз: 4С2*4 = 24
итого: (192+144+16+24)*12С5 = 376*12С5
4) вошло 6 карт, число удачных комбинаций:
а) пара + туз + 3 любых: 4*4*(6С3-3*4) = 128
б) пара + 2 туза + 2 любых: 4*4С2*(6С2-3) = 288
в) пара + 3 туза + любая: 4*4С3*6 = 96
г) пара + 4 туза: 4*1 = 4
д) 2 пары + туз + любая: 4С2*4*4 = 96
е) 2 пары + 2 туза: 4С2*4С2 = 36
итого: (128+288+96+4+96+36)*12С4 = 648*12С4

5) вошло 7 карт, число удачных комбинаций:
а) пара + туз + 4 любых: 4*4*(6С4-3*(4С2-2)-3С2) = 0, т.к. 4 из 6 половин = пара
б) пара + 2 туза + 3 любых: 4*4С2*(6С3-3*4) = 192
в) пара + 3 туза + 2 любых: 4*4С3*(6С2-3) = 192
г) пара + 4 туза + любая: 4*1*6 = 24
д) 2 пары + туз + 2 любых: 4С2*4*(4С2-2) = 96
е) 2 пары + 2 туза + любая: 4С2*4С2*4 = 144
ж) 2 пары + 3 туза: 4С2*4С3 = 24
з) 3 пары + туз: 4С3*4 = 16
итого: (192+192+24+96+144+24+16)*12С3 = 688*12С3
6) вошло 8 карт, число удачных комбинаций:
а) пара + туз + 5 любых: 0
б) пара + 2 туза + 4 любых: 0
в) пара + 3 туза + 3 любых: 4*4С3*(6С3-3*4) = 128
г) пара + 4 туза + 2 любых: 4*1*(6С2-3) = 48
д) 2 пары + туз + 3 любых: 4С2*4*(4С3-2*2) = 0, т.к. 3 из 4 половин = пара
е) 2 пары + 2 туза + 2 любых: 4С2*4С2*(4С2-2) = 144
ж) 2 пары + 3 туза + любая: 4С2*4С3*4 = 96
з) 2 пары + 4 туза: 4С2*1 = 6
и) 3 пары + туз + любая = 4С3*4*2 = 32
к) 3 пары + 2 туза = 4С3*4С2 = 24
итого: (128+48+144+96+6+32+24)*12С2 = 478*12С2
7) вошло 9 карт, любые комбинации удачны: 12С9*12
8) вошло 10 карт, любые комбинации удачны: 12С10

Искомая вероятность найдется: число удачных / общее число комбинаций * 100%.

p = (16*12С7+120*12С6+376*12С5+648 *12С4+478*12С2+12*12С9+12С10)/24C10 * 100% = 47.30%

Задача 2.4. Вероятность наличия на руках комбинации Т 10 К Д любой одинаковой(!) масти при игре в темную.
По аналогии с задачей 2.3. Найдем число комбинаций, являющихся для нас благоприятными. Всего имеем дело с 16 интересующими нас картами.

1) вошло 4 карты, число комбинаций (Т10КД): 4*8С6
2) вошло 5 карт, число комбинаций (Т10КД + любая): 4*12С1*8С5 = 48*8С5
3) вошло 6 карт, число комбинаций (Т10КД + 2 любых): 4*12С2*8С4 = 264*8C4
4) вошло 7 карт, число комбинаций (Т10КД + 3 любых): 4*12С3*8С3 = 880*8C3
5) вошло 8 карт, число комбинаций:
а) Т10КД + 4 любых: 4*(12С4-3) = 1968
б) Т10КД + Т10КД: 4С2 = 6
итого: (1968+6)*8С2 = 1974*8С2
6) вошло 9 карт, число комбинаций:
а) Т10КД + 5 любых: 4*(12С5-3*8) = 3072
б) Т10КД + Т10КД + любая: 4С2*8 = 48
итого: (3072+48)*8C1 = 24960
7) вошло 10 карт, число комбинаций:
а) Т10КД + 6 любых: 4*(12С6-3*8С2) = 3360
б) Т10КД + Т10КД + 2 любых: 4С2*8С2 = 168
итого: 3360+168 = 3528

Искомая вероятность найдется: число удачных / общее число комбинаций * 100%.
p = 7.87%.

На этом самую обширную часть работы закончили. Переходим к чему-по-проще, а именно к игре в светлую.

Игра в светлую.

Карты, пришедшие на руки после раздачи, игроку известны, поэтому рассчитывать вероятности вхождения тех или иных комбинаций в расклад смысла не имеет. Остается вопрос о прикупе. Всего имеется 17 (24-7) неизвестных карт, 3 из которых случайным образом входят в прикуп. Вариантов прикупа 17С3. Формулы практически идентичны формулам прикупа для игры в темную, с той лишь разницей, что число 24 заменяется числом 17. Все же формулы приведены ниже, для полноты картины.

Вероятность докупить 1 карту из n нужных (n>=1).
p(1,n) = (17C3 - (17-n)C3) / 17C3 * 100%, где n=1..17. (3.1)
для n = 15, 16, 17 вероятность равна 100%, т.к. вычитаемое вырождается.

Вероятность докупить 2 карты из n нужных (n>=2).
p(2,n) = (17C3 - n*(17-n)C2 - (17-n)C3) / 17C3 * 100%, где n=2..17. (3.2)
для n = 16, 17 вероятность равна 100%, т.к. вычитаемые вырождаются.
Вероятность докупить 3 карты из n нужных (n>=3).
p(2,n) = nC3/17C3*100%, где n=3..17. (3.2)

В довесок к вышеприведенному материалу решим еще несколько задач.

Задача 3.1. У игрока на руках тузов нет. Вероятность докупить любого из 4 тузов.
Речь идет о вероятности докупить хотя бы 1 любую карту из 4.
p(1,4) = (17C3-13C3)/17C3*100% = 57.94%.

Задача 3.2. У игрока на руках нет ни одной половины. Вероятность докупить любую пару (хваль).
Всего есть 4 пары, каждая занимает места 2 карт прикупа. Третья карта может быть выбрана из 15 оставшихся.
p = 4*15/17C3*100% = 8.82%.

Задача 3.3. У игрока на руках нет ни тузов, ни половин. Вероятность докупить любую пару (хваль) с любым из тузов. 
Всего есть 4 пары и 4 туза, что дает 16 возможных комбинаций взаимного расположения.
p = 4*4/17C3*100% = 2.35%.

Задача 3.4. У игрока на руках есть 4 половины, но нет тузов. Вероятность докупить любого туза с любой из оставшихся половин.
p = 4*4*15/17C3 = 35.29%.

Ниже рассмотрим более обыденный случай, когда есть чем брать первую взятку и есть несколько половин. Удастся ли игроку докупить до пары?

Задача 3.5. У игрока на руках есть 1 половина. Вероятность докупить до пары недостающую половину и/или докупить пару.
Вероятность докупить половину: p1 = p(1,1) = (17C3-16C3)/17C3*100% = 17.65%.
Вероятность докупить пару: p2 = 3*14/17C3*100% = 6.18%.
Искомая вероятность: p = p1 + p2 = 23.82%.

Задача 3.6. У игрока на руках есть 2 половины. Вероятность докупить до пары хотя бы одну недостающую половину и/или докупить пару.
Вероятность докупить половину: p1 = p(1,2) = (17C3-15C3)/17C3*100% = 33.09%.
Вероятность докупить пару: p2 = 2*13/17C3*100% = 3.82%.
Искомая вероятность: p = p1 + p2 = 36.91%.

Задача 3.7. У игрока на руках есть 3 половины. Вероятность докупить до пары хотя бы одну недостающую половину и/или докупить пару.
Вероятность докупить половину: p1 = p(1,3) = (17C3-14C3)/17C3*100% = 46.47%.
Вероятность докупить пару: p2 = 1*12/17C3*100% = 1.76%.
Искомая вероятность: p = p1 + p2 = 48.24%.

Задача 3.8. У игрока на руках есть 4 половины. Вероятность докупить до пары хотя бы одну недостающую половину.
p(1,4) = (17C3-13C3)/17C3*100% = 57.94%

(*)Все приведенные выше значения вероятностей были проверены с помощью программы-переборщика. Значения, полученные путем усреднения данных одного миллиона раздач, не расходятся с расчетными (полученными по комбинаторным формулам) более чем на 0.1%.
